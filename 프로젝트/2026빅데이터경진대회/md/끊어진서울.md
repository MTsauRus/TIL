「끊어진 서울」 — 시각화 5종 상세 설계
📐 핵심 분석 개념 한 줄

지상철도는 **선형 장벽(linear barrier)**이다. 시민이 이 장벽을 건너려면 **횡단점(crossing)**을 거쳐야 하며, 횡단점이 듬성듬성할수록 우회 보행거리가 폭증한다. 이 우회 페널티를 정량화·지도화한다.

시각화 1. 단절 지도 (Disconnection Heatmap)
무엇을 보여주는가
서울 전체 지도 위에서, 지상철도 양옆 500m 버퍼 안의 보행자가 반대편으로 가려면 평균 몇 m를 우회해야 하는가를 색으로 표현. 우회가 심한 구간일수록 빨갛게 빛나는 "출혈선"으로 보임.
정확히 어떻게 그려지는가 (단계별)
[1단계] 지상철도 선분(LineString)을 OSM에서 추출
└ railway=rail, bridge!=yes, tunnel!=yes 인 구간만

[2단계] 선분을 50m 간격으로 점(sample point)으로 분해
└ 약 2,000~3,000개 샘플점 생성

[3단계] 각 샘플점에서 양옆 200m 떨어진 곳에
가상의 "출발점 N"과 "목표점 S"를 만든다
(=철도 바로 옆에 사는 가상 시민)

[4단계] N→S로 가려면 보행 네트워크에서 최단경로를 계산
└ 직선거리는 400m
└ 실제 보행거리는? 600m? 1,500m? 3,000m?

[5단계] 우회비율 = 실제거리 / 직선거리 (1.0이면 단절 없음, 5.0이면 5배 돌아감)
└ 이 값을 샘플점에 색으로 매핑

[6단계] 샘플점들을 선으로 이으면 → 철도를 따라 흐르는 "출혈선"
결과물 이미지 (말로 묘사)
서울 지도 위에 검은 행정동 경계가 옅게 깔리고, 지상철도 노선만 굵게 빛난다. 단절 적은 구간(예: 신도림 일대 횡단 인프라 충실)은 연한 청록, 단절 심한 구간(예: 영등포-신길 일부, 중랑천 근접 구간 등)은 선홍색으로 빛남. 마치 동맥류처럼 빨간 점이 부풀어 있는 곳이 시각적 핵심.
사용 데이터
데이터출처URL지상철도 선형 (railway=rail)OSM (보조)https://www.openstreetmap.org / Overpass API보행 네트워크 (walkable network)OSM (보조)OSMnx 라이브러리로 자동 다운로드서울 행정동 경계서울 데이터 허브 [필수]행정안전부 / data.seoul.go.kr 동 경계 (또는 stat.kr SGIS)서울시 보도육교 위치현황서울 데이터 허브 [필수]https://data.seoul.go.kr/dataList/OA-22353/F/1/datasetView.do서울시 육교 공간정보서울 데이터 허브 [필수]https://data.seoul.go.kr/dataList/OA-21210/S/1/datasetView.do
구현 방법
pythonimport osmnx as ox
import geopandas as gpd
import networkx as nx
from shapely.geometry import LineString, Point

# 1. 보행 네트워크 (서울 전체)

G = ox.graph_from_place("Seoul, South Korea", network_type="walk")

# 2. 지상철도 추출

rails = ox.features_from_place(
"Seoul, South Korea",
tags={"railway": "rail"}
)
above_ground = rails[(rails["bridge"] != "yes") & (rails["tunnel"] != "yes")]

# 3. 50m 간격 샘플

sample_points = []
for line in above_ground.geometry:
for d in range(0, int(line.length), 50):
sample_points.append(line.interpolate(d))

# 4. 우회비율 계산

results = []
for p in sample_points: # 철도 진행방향의 수직 양쪽 200m
n_pt, s_pt = perpendicular_offset(p, line, 200)
n_node = ox.nearest_nodes(G, n_pt.x, n_pt.y)
s_node = ox.nearest_nodes(G, s_pt.x, s_pt.y)
try:
walk_len = nx.shortest_path_length(G, n_node, s_node, weight="length")
ratio = walk_len / 400
except nx.NetworkXNoPath:
ratio = float("inf")
results.append((p, ratio))

시각화 4. 횡단점 간격 단면도 (Crossing Gap Strip Chart)
무엇을 보여주는가
지상철도 노선을 수평 띠로 펼쳐놓고, 그 위에 횡단점(육교·건널목·교차도로 등)을 점으로 찍는다. 점 사이의 빈 공간 = "여기서는 못 건넌다." 어느 노선의 어느 구간에 틈이 큰지 한눈에.
그리는 방법

노선별로 위에서 아래로 정렬 (1호선/경의중앙선/공항철도/등등)
x축 = 노선 시점에서의 거리 (0 ~ 노선 길이)
y축 = 노선 이름
점 = 횡단 가능 지점 (육교 + 도로 교차점)
점 사이의 빈 공간이 빨간 음영으로 채워져서 "이 구간은 1.2km째 못 건너고 있다" 강조

사용 데이터
데이터출처URL서울시 육교 공간정보서울 데이터 허브 [필수]https://data.seoul.go.kr/dataList/OA-21210/S/1/datasetView.do서울시 보도육교 위치현황서울 데이터 허브 [필수]https://data.seoul.go.kr/dataList/OA-22353/F/1/datasetView.do서울시 육교정보 (자치구별)서울 데이터 허브https://data.seoul.go.kr/dataList/OA-22361/F/1/datasetView.do서울시 지하철역 연계 지하도 공간정보서울 데이터 허브 [필수]https://data.seoul.go.kr/dataList/OA-21213/S/1/datasetView.do도로 ↔ 철도 교차점OSM(도로/철도 GeoDataFrame intersect)
