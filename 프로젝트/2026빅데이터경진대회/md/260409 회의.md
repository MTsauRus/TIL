## 유호준

- 처음엔 신호가 짧아서 사고난다 -> 아님
- 골목길 보행사고가 많음.
- 큰도로 사고 안남.
- 발표전략: 평소에 횡단보도로 어그로끌고, 그런데말입니다 하고 골목길에서 더 많이난다 전환

## 이정태

- 24시간 비욘드 조닝 도입을 위한 융복합 최적 입지 타겟팅
- 주거 / 산업용으로 엄격하게 분리되는 것에 대한 문제의식
- 인구공동화현상 섞을 수 있음
- 특정 지역이 낮에는 붐비고 밤에는 유령도시가 됨. 혹은 그 반대.
- 이게 비효율적 -> 이런 데이터를 뽑아서 유연하게 대처해보자
- 관련 데이터 ~
- 다기능 복합지역 / 용도지역변경 / 인구통계(주야간) 섞어서 지역별 인사이트 제공
- 어떻게 하면 효율적으로 도시공간을 활용할 수 있는가
- 다양성 점수 매겨서, 낮은 곳 / 높은 곳 인구통계와 유의미한 상관관계(주거지규제를 푼다거나, 하는 방식으로 도시경제활성화에 기여할 수 있음)
  - 이 과정에서 히트맵 제시.
- 밤 / 낮 유령도시 없이 잘 나눠보자.

## 심재현

- 기반시설 입체화 중심
- 시각화: 소음민원데이터, 건축물 노후도, 공원 녹지율, 문화시설 분석
- 지상 철도의 지역 발전 저해 정도를 시각화할 수 있음
- 경제/환경적 효과 -> 데이터왜곡 가능성 있음.
- 데크위에 보행일상권 확보할 수 있다 -> 도시공간 효율적으로 활용
  - 정책기여가능
- 소음측정정보 등 관련데이터
- 철도지하화 관련기사가 많음
- 상부활용 -> 입체공원 -> 녹지확대가능
- 입체공원 입지 선정을 해주자.
  - 서울시에는 밀지만 국토부에서는 반대하는 특징이 있음.
- 공간활용방안 중심
- 문제점: 지하화 -> 싱크홀, 땅꺼짐 문제 가능성 있음.

### 수변중심 공간재편

- 2040 도시건축 디자인혁신 계획이 있음
- 이거에 부합함
- 하천중심, 그레이트한강 충족 가능
