- List
  - 순서 및 중복 저장
  - boolean add(E e): 마지막에 삽입
  - void add(idx, e): 인덱스에 삽입
  - set(idx, e): 인덱스에 수정
  - contains(obj): 포함여부, true/false
  - get(idx)
  - isEmpty()
  - size()
  - clear()
  - remove()
    - idx, obj 둘 다 가능

### ArrayList

- 삽입/삭제 시 인덱스를 당김
- 잦은 수정 시 사용 x.
- 동일 객체 저장 가능
- null 저장 가능

### Vector

- ArrayList의 sync버전

### LinkedList

- 잦은 수정 삭제 시 이거 사용

## Set

- 순서 중복 저장 x
- add()
- contains()
- isEmpty()
- iterator(): Iterator<E> 객체 리턴
  - hasNext()
  - next()
  - remove(): next로 가져온 것 set에서 제거
- hashCode(), equals() 메서드 재정의해야 함
- 아래처럼 쓰면댐

```java
    Iterator<String> iterator = set.iterator();
    while (iterator.hasNext()) {
        String e = iterator.next();
        if (e.equals("hi")) {
            iterator.remove();
        }
    }
```

## Map

- key, value로 이루어진 Entry 저장
- put(k, v)
- containsKey()
- containsValue()
- entrySet(): return Set`<Map.Entry<k, v>>`
- get(K)
- keySet(), values()

### HashMap

- 키 중복저장 x

```java
Map<k, v> map = new HashMap<>();
Set<Entry<String, Integer>> entrySet = map.entrySet(); // 맵 안에 있는 엔트리를 세트로 가져옴
Iterator<Entry<String, Integer>> iterator = entrySet.iterator();
```

## TreeSet, TreeMap

- 이진 트리 구조의 Set, Map
- 부모 왼쪽에 작은거, 부모 오른쪽에 큰거
- Map은 obj에 엔트리 저장
- first()
- last()
- lower()
- higher()
- floor()
- ceiling()
- pollFirst(): 젤 작은거 뽑고 제거
- pollLast()
- descendingIterator()
- descendingSet()
