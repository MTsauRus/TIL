# 조선 전기 정치

## 조선 건국 흐름 ⭐⭐⭐

위화도 회군 (1388, 이성계)
→ 과전법 실시 (1391, 토지 개혁)
→ 정몽주 제거 (1392, 이방원)
→ 조선 건국 (1392, 이성계+정도전)

### 정몽주 vs 정도전

| 구분   | 정몽주                               | 정도전                                                        |
| ------ | ------------------------------------ | ------------------------------------------------------------- |
| 노선   | 고려 왕조 유지                       | 새 왕조 건국                                                  |
| 시조   | "이 몸이 죽고 죽어 일백번 고쳐 죽어" | -                                                             |
| 저서   | -                                    | **《조선경국전》**(최초 사찬법전), **《불씨잡변》**(불교비판) |
| 정치론 | -                                    | **재상정치** (신권 중심)                                      |
| 결말   | 이방원에게 피살 (선죽교)             | 1차 왕자의 난 때 이방원에게 제거                              |

---

## 왕권 강화 vs 신권 강화 ⭐⭐⭐

| 왕권 강화                          | 신권(재상권) 강화                      |
| ---------------------------------- | -------------------------------------- |
| **6조직계제** (태종·세조)          | **의정부서사제** (세종·성종)           |
| **의금부** (왕 직속 사법기관)      | **사간원** (간쟁·서경)                 |
| **승정원** (왕 비서기관, 왕명출납) | **사헌부** (관리 감찰, 탄핵)           |
| 집현전 폐지 (세조)                 | **홍문관** (경연, 왕 자문) — 성종 설치 |
| 경연 폐지 (세조)                   | **3사** = 사간원+사헌부+홍문관 (언론)  |

> 💡 **조선 3사** = 언론기관 (고려 삼사는 회계기관 — 완전히 다름!)

---

## 태조 (1392~1398)

- **조선 건국** (1392), 한양 천도 (1394)
- **숭유억불** 정책
- **과전법** 시행: 전·현직 관리 모두에게 경기 지역 토지 지급
- **의흥삼군부** 설치: 군사 기구
- **천상열차분야지도**: 고구려 천문도 바탕으로 제작한 천문도 — 태조 때 돌에 새김 ⭐
- **정도전** 주도로 한양 도성·경복궁 설계, 4대문 이름 지음

---

## 태종 (1400~1418) ⭐⭐⭐

- **6조직계제**: 의정부 배제, 6조가 왕에게 직접 보고
- **사병 혁파**: 공신·왕족 사병 몰수 → 군사력 왕에게 집중
- **호패법**: 16세 이상 남자 신분증 → 인구·군역·세금 파악
- **신문고 설치**: 억울한 백성 직소 (실효성은 낮았음)
- **외척 제거**: 처가 민씨 세력 숙청
- **사간원 독립**: 언론기관 강화
- **양전사업**: 토지 조사로 재정 확충
- **혼일강리역대국도지도 (1402)** ⭐⭐⭐: 현존 동양 최고(最古) 세계지도
- **주자소 설치·계미자 주조**: 금속활자 인쇄소

---

## 세종 (1418~1450) ⭐⭐⭐ 최다 출제

### 정치·제도

- **의정부서사제**: 6조 → 의정부 심의 → 왕 재가 (신권 강화)
- **집현전 강화**: 학문 연구·인재 양성 (세조 때 폐지됨)
- **경연 활성화**: 왕과 신하가 함께 학문 토론

### 훈민정음 ⭐⭐⭐

- **창제 1443 / 반포 1446**
- 창제 목적: 백성 교화, 문자 불편 해소
- 반대: **최만리** 등 집현전 일부 학자 (중국 문화 침해 우려)
- 이후 편찬: **《용비어천가》《월인천강지곡》**

### 과학·기술 ⭐⭐

| 발명품       | 내용                    | 제작자 |
| ------------ | ----------------------- | ------ |
| **측우기**   | 세계 최초 강우량 측정기 | 장영실 |
| **자격루**   | 자동 물시계             | 장영실 |
| **앙부일구** | 해시계                  | 장영실 |
| **혼천의**   | 천체 관측기             |        |
| **칠정산**   | 우리 실정에 맞는 역법서 |        |

> 💡 **장영실**: 동래현 노비 출신 → 세종이 발탁

### 편찬 사업 ⭐⭐

| 서적                | 분야                              |
| ------------------- | --------------------------------- |
| **《농사직설》**    | 농업 (우리 풍토 맞는 농법)        |
| **《향약집성방》**  | 의학 (국산 약재 정리)             |
| **《의방유취》**    | 의학 백과                         |
| **《삼강행실도》**  | 윤리 (충·효·열 사례집, 그림 포함) |
| **《동국정운》**    | 표준 한자음 정리                  |
| **《고려사》 착수** | 완성은 문종 때                    |

### 4군 6진·대외 관계 ⭐⭐

- **4군**: 최윤덕 — 압록강 유역 여진 몰아냄
- **6진**: 김종서 — 두만강 유역 확보 → 현재 한반도 국경선 완성
- **쓰시마(대마도) 정벌 (1419)**: 이종무 파견, 왜구 소탕
- **계해약조 (1443)**: 3포(부산포·제포·염포) 개항, 세견선 50척 허용

### 조세 — 공법 ⭐⭐⭐

- 기존 1결 30두 고정 → 불합리
- **전분6등법**: 토지 비옥도에 따라 6등급 (풍흉과 무관)
- **연분9등법**: 그 해 풍흉에 따라 9등급
- 두 가지 결합 → 세금 체계화 (최저 4두 ~ 최고 20두)

## 문종 (1450~1452)

- **《고려사》 완성** (세종 착수 → 문종 완성)
- **《동국병감》**: 고조선~고려 전쟁사
- **화차 개량**: 화기 무기 정비
- 건강 악화 단명 → 어린 단종 즉위 → 계유정난 배경

## 단종·세조 (앞 세조 편 참조)

- 계유정난(1453) → 세조 즉위(1455)
- 사육신 단종 복위 운동(1456) 실패
- **직전법(1466)**: 현직 관리에게만 토지 지급
- **6조직계제 부활, 집현전·경연 폐지**
- **진관체제 정비**: 지방 군사 방어 체계
- **《경국대전》 편찬 착수**: 완성은 성종
- **간경도감**: 불경 번역 (세조는 불교 우호적)
- **원각사지 10층 석탑**: 세조 때 건립 ⭐ (고려 경천사지 10층 석탑과 구분!)

## 예종 (1468~1469)

- 단명, 별 업적 없음
- **남이의 옥**: 공신 세력 견제

---

## 성종 (1469~1494) ⭐⭐⭐

### 왕권·제도

- **《경국대전》 완성 (1485)** ⭐⭐⭐: 조선 기본 법전 완성
- **의정부서사제 유지**: 왕과 신하 협치
- **홍문관 설치** ⭐⭐⭐: 경연 담당, 왕 자문 → 3사 체제 완성
- **경연 활성화**: 하루 3번 경연 실시할 정도

### 사림 중앙 진출 ⭐⭐⭐

- **김종직 등 사림파** 대거 등용
- 훈구 세력 견제 목적 → 3사(언론기관)에 배치
- 사림이 훈구 비리 공격 → 갈등 심화 → 후에 사화로 이어짐

### 편찬 사업

- **《동국통감》**: 고조선~고려 통사
- **《동문선》**: 우리나라 문학 작품 선집
- **《동국여지승람》**: 지리서
- **《악학궤범》**: 음악 이론서
- **유향소 부활**: 향촌 자치 허용 (세조 때 폐지했던 것)

---

## 연산군 (1494~1506) ⭐⭐⭐

- **무오사화 (1498)**: 김일손 사초에 **김종직 《조의제문》** 수록 → 사림 탄압
  - 《조의제문》: 항우에게 죽은 의제 추모 = 사실상 단종 죽인 세조 비판
  - 이미 죽은 김종직 **부관참시**
- **갑자사화 (1504)**: 어머니 **폐비 윤씨** 사사 사건 발각 → 대규모 숙청
  - 사림뿐 아니라 훈구도 피해
- **폭정**: 성균관을 유흥장으로, 경연 폐지, 언론 탄압
- **중종반정 (1506)**: 훈구 세력이 연산군 폐위

---

## 중종 (1506~1544) ⭐⭐⭐

### 조광조 개혁 (기묘사화 1519)

| 개혁 내용            | 설명                                     |
| -------------------- | ---------------------------------------- |
| **현량과 실시**      | 추천제로 사림 대거 등용                  |
| **소격서 폐지**      | 도교 행사 기관 폐지 → 성리학 강화        |
| **위훈 삭제** ⭐⭐⭐ | 중종반정 가짜 공신 삭제 주장 → 훈구 격분 |
| **향약 보급**        | 지방 사림 세력 확대                      |

- **주초위왕(走肖爲王) 사건**: 나뭇잎에 "조씨가 왕이 된다" 조작
- 조광조 등 사림 대거 처형·유배 → **기묘사화**
- **《신증동국여지승람》**: 지리서 재편찬

## 명종 (1545~1567)

- **을사사화 (1545)**: 대윤(윤임, 인종 외척) vs 소윤(윤원형, 명종 외척)
  - 인종 사망 → 명종 즉위 → 소윤 승리, 대윤 몰락
- **문정왕후 수렴청정**: 명종의 어머니, 실권 장악
- **윤원형** 전횡
- **임꺽정의 난**: 황해도 도적 — 수취 체제 모순 폭발 상징
- **비변사 강화**: 왜구·여진 대비 군사기구 → 후에 핵심 국정기관으로 성장

## 훈구 vs 사림 ⭐⭐⭐

| 구분          | 훈구               | 사림                     |
| ------------- | ------------------ | ------------------------ |
| **기원**      | 세조 계유정난 공신 | 길재→김종직 계보         |
| **학문**      | 실무·행정          | 성리학 이념              |
| **경제 기반** | 대농장             | 지방 중소 지주           |
| **진출 경로** | 공신 책봉·음서     | 과거                     |
| **활동 무대** | 중앙 조정          | 지방 → 성종 때 중앙 진출 |
| **정치 성향** | 왕권과 타협        | 왕도정치·도덕·명분       |
| **근거지**    | 중앙               | 서원·향약                |

### 사림의 계보

정몽주 (절의파)
↓
길재 (낙향·은둔)
↓
김숙자
↓
김종직 (성종 때 중앙 진출)
↓
김굉필·정여창·김일손
↓
조광조 (중종)
↓
이황·이이 (명종·선조)

### 사림이 사화를 4번 당하고도 살아남은 이유

> **서원·향약**으로 지방 기반 꾸준히 유지
> → 선조 때 사림 최종 집권 성공

# 조선 전기 외교

## 대명 외교 기조 — 친명사대 ⭐⭐

- 조선 건국의 정통성을 명나라로부터 인정받음
- **사대교린**: 명에는 사대(事大), 여진·일본에는 교린(交隣)
- 조공·책봉 체제 유지
- 명나라 연호 사용, 명의 문물 수용
- 임진왜란 때 명이 파병 → **재조지은(再造之恩)** 의식 강화

---

## 임진왜란 (1592~1598) ⭐⭐⭐

### 배경

- 일본 **도요토미 히데요시** 전국 통일 후 조선 침략
- 조선 200년 평화로 군사력 약화

### 전쟁 초기

- 왜군 파죽지세 — 부산 상륙 후 20일 만에 한양 함락
- 선조 의주로 피난, 명나라에 지원 요청

### 3대첩 ⭐⭐⭐

| 전투            | 인물       | 내용                                    |
| --------------- | ---------- | --------------------------------------- |
| **한산도 대첩** | **이순신** | 학익진 전술, 왜 수군 대파 — 제해권 장악 |
| **진주 대첩**   | **김시민** | 진주성 방어 성공                        |
| **행주 대첩**   | **권율**   | 행주산성, 행주치마 설화                 |

### 의병 ⭐⭐

| 인물       | 지역                         |
| ---------- | ---------------------------- |
| **곽재우** | 경상도 (홍의장군)            |
| **조헌**   | 충청도 (700의총 — 금산 전투) |
| **고경명** | 전라도                       |
| **정문부** | 함경도                       |

- 승병: **서산대사(휴정)**, **사명대사(유정)**

### 명나라 참전

- **이여송** 이끄는 명군 파병
- **평양성 수복** → 이후 벽제관 전투에서 패배

### 휴전기 군사 개혁 ⭐⭐⭐

- **훈련도감 설치**: 직업 군인 — 포수·사수·살수 **삼수병** (조총 도입)
- **속오군 편성**: 양반~천민 포함한 지방 방위군
- **비변사 강화**: 군사 최고 기관으로 성장

### 정유재란 (1597)

- 협상 결렬 → 왜군 재침
- **명량 해전**: 이순신 13척으로 133척 왜선 격파 ⭐⭐⭐
- **노량 해전 (1598)**: 이순신 전사, 왜군 완전 퇴각

### 임진왜란 결과

- 조선: 인구 감소, 토지 황폐화, 문화재 소실, 경복궁 소실
- 일본: 도요토미 정권 붕괴 → 도쿠가와 에도 막부
- 명나라: 국력 소진 → 여진(후금) 성장 기회

---

## 광해군 (1608~1623) ⭐⭐⭐

### 업적

- **중립 외교**: 명 쇠퇴 + 후금 성장 → 줄타기 외교
- **강홍립 투항 사건**: 명의 요청으로 파병했으나 후금에 의도적 투항 → 실리 외교
- **대동법 실시 (경기도 선 시행)** ⭐⭐⭐
  - 공납(특산물) → 쌀·베·동전으로 통일
  - 방납 폐단 해결 목적
  - 이후 숙종 때 전국 확대
- **《동의보감》 완성**: 허준 저술 (선조 때 시작 → 광해군 때 완성) ⭐⭐
- **전후 복구**: 토지 조사, 인구 재편

### 인조반정 (1623) ⭐⭐⭐

**배경**

- 서인 세력이 광해군의 중립 외교를 **친명사대 배반**으로 비판
- **영창대군 사사**, **인목대비 폐비** — 패륜 행위로 공격
- 훈구·서인 연합으로 광해군 폐위

> 💡 실제론 서인이 권력 탈취 목적 + 명분으로 도덕 문제 활용

---

## 인조 (1623~1649) ⭐⭐⭐

### 친명 외교 복원

- **기유약조 (1609, 광해군)**: 임진왜란으로 단절된 일본과 국교 재개
  - 동래 부산포 1포만 개항, 세견선 20척
  - 참고: 계해약조(세종)는 3포·50척

### 정묘호란 (1627)

- 후금이 **인조반정 = 친명 정권** 등장에 위협 느껴 침입
- 인조 **강화도 피신**
- **형제 관계** 맺고 강화 (후금 형, 조선 동생)

### 병자호란 (1636~1637) ⭐⭐⭐

- 후금이 **청**으로 국호 변경 → **군신 관계** 요구
- 조선 거부 → 청 태종 직접 침입
- 인조 **남한산성**으로 피신, 45일 항전
- **삼전도의 굴욕 (1637)**: 인조가 청 태종에게 **삼배구고두례** (3번 절하고 9번 머리 조아림)
- **소현세자·봉림대군** 청나라 볼모
- 이후 **북벌론** 등장

### 효종 (1649~1659)

- **북벌론** ⭐⭐: 청나라에 복수, 명나라 의리 지키자는 주장
  - 실제론 왕권 강화 목적도 있음
  - **이완** 등과 군비 확충
  - 현실적으로 실행 못함
- **나선 정벌**: 청의 요청으로 러시아(나선) 정벌에 조총 부대 파견 ⭐

# 조선 전기 경제

## 토지제도 흐름 ⭐⭐⭐

과전법(태조 1391) → 직전법(세조 1466) → 관수관급제(성종) → 직전법 폐지

| 제도            | 시기   | 내용                                                        |
| --------------- | ------ | ----------------------------------------------------------- |
| **과전법**      | 태조   | 전·현직 관리 모두 지급, 경기 지방만, 원칙상 세습 불가       |
|                 |        | but 수신전·휼양전으로 사실상 세습 → 토지 부족               |
| **직전법**      | 세조   | **현직** 관리에게만 지급, 수신전·휼양전 폐지                |
| **관수관급제**  | 성종   | 국가가 직접 세금 거둬 관리에게 지급 → 관리의 직접 수취 금지 |
| **직전법 폐지** | 16세기 | 관리에게 녹봉만 지급 → 토지 지급 완전 소멸                  |

---

## 수취 체제 ⭐⭐⭐

### 전세 (토지세)

- 기존: 1결당 30두 고정 → 불합리
- **공법 (세종)**: 전분6등법 + 연분9등법 → 최저 4두 ~ 최고 20두

### 공납 (특산물)

- 지역 특산물을 현물로 납부
- **방납의 폐단** ⭐⭐⭐
  - 대행업자(방납인)가 대신 납부하고 엄청난 이윤 착취
  - 백성 부담 가중
  - → 해결책: **대동법** (광해군~숙종)

### 역 (노동력)

- 16~60세 남성 군역·요역 의무
- **대립 폐단**: 돈 주고 다른 사람 대신 보냄
- **방군수포 폐단**: 군역 대신 포를 바침 → 군사력 약화
- → 해결책: **균역법** (영조)

> 💡 이런 수취 체제 모순 → **임꺽정 같은 도적** 출현 배경

---

## 대동법 ⭐⭐⭐

- **광해군** 때 경기도 선 시행 (이원익 건의)
- 공납(특산물) → **쌀·삼베·동전**으로 통일
- 가구 단위 → **토지 결수** 기준으로 부과 (빈부 불균형 완화)
- **공인(貢人)** 등장: 대동미 받아 물건 납품하는 상인 → 상품 경제 발달 촉진
- 전국 확대: 인조~숙종 (약 100년에 걸쳐)

## 신분제 ⭐⭐⭐

양반 — 중인(서얼 포함) — 상민(양인) — 천민

| 신분           | 내용                                       |
| -------------- | ------------------------------------------ |
| **양반**       | 문반+무반, 과거·음서로 관직, 군역 면제     |
| **중인**       | 기술관(역관·의관·화원 등), 서얼(양반 서자) |
| **상민(양인)** | 농민·수공업자·상인, 군역·납세 의무         |
| **신량역천**   | 양인이지만 천한 일 (조례·나장·봉수군 등)   |
| **천민**       | 노비(매매·상속·증여 가능), 백정·광대 등    |

> 💡 **고려 백정** = 일반 양민 / **조선 백정** = 천민 (도살업) — 완전히 다름!
> 💡 신분제 폐지: **1894년 갑오개혁**

# 조선 전기 문화

## 15세기 문화 — 민족 자주 문화 ⭐⭐

### 편찬·사상

- **훈민정음 (1446)**: 세종
- **칠정산**: 독자 역법 — 중국·아라비아 역법 참고
- **향약집성방·농사직설**: 자주적 실용 서적

### 역사·지리

- **《고려사》**: 기전체, 문종 완성
- **《고려사절요》**: 편년체
- **《동국통감》**: 성종 때 완성, 고조선~고려 통사
- **《동국여지승람》**: 성종 때 지리서

### 지도

- **혼일강리역대국도지도 (1402)** ⭐⭐⭐
  - 태종 때 제작, 현존 동양 최고(最古) 세계지도
  - 유럽·아프리카까지 표현
- **천상열차분야지도** ⭐⭐
  - 태조 때 돌에 새긴 천문도
  - 고구려 천문도 바탕으로 제작
  - 세계에서 두 번째로 오래된 천문도

### 미술·건축

- **안견 《몽유도원도》**: 안평대군의 꿈을 그린 그림 ⭐⭐
- **강희안 《고사관수도》**: 선비가 물을 바라보는 그림
- **분청사기**: 고려 청자 → 조선 특유의 소박·역동적 도자기 ⭐
- **경복궁·창덕궁** 등 궁궐, **한양 도성** 건축
- **원각사지 10층 석탑** ⭐⭐
  - 세조 때 건립 (원각사 터, 지금의 탑골공원)
  - 고려 경천사지 10층 석탑 양식 계승 — 시험에서 혼동 주의!

## 16세기 문화 — 사림 성리학 문화 ⭐⭐⭐

### 성리학 — 이황 vs 이이 ⭐⭐⭐

| 구분      | **이황 (퇴계)**                  | **이이 (율곡)**                  |
| --------- | -------------------------------- | -------------------------------- |
| 시대      | 명종·선조                        | 선조                             |
| 출신      | 경상도 (영남학파)                | 경기도 (기호학파)                |
| 주요 저서 | **《성학십도》** (선조에게 바침) | **《성학집요》** (선조에게 바침) |
|           | **《동호문답》**                 | **《동호문답》**                 |
|           | **《주자서절요》**               | **《격몽요결》** (초학자용)      |
| 철학      | **이(理) 중심** — 이기이원론     | **기(氣) 중심** — 이기일원론     |
| 성향      | 원칙적·도덕적, 현실보다 이념     | **현실 개혁적**, 실용적          |
| 주장      | 수양·도덕 강조                   | **10만 양병설** (현실 국방)      |
| 영향      | 일본 성리학에 큰 영향 ⭐         | 조선 후기 실학에 영향            |
| 서원      | **도산서원** (안동)              | **자운서원**                     |
| 화폐      | 현재 **1000원권**                | 현재 **5000원권**                |

> 💡 **이황** = 원칙주의, 도덕 → 일본까지 영향
> **이이** = 현실개혁, 실용 → 10만 양병설 (임진왜란 예측)

### 서원 ⭐⭐⭐

- 사림의 지방 교육·제사 기관
- **백운동 서원**: 중종 때 **주세붕** 설립 (최초 서원)
- **소수서원**: 이황이 명종에게 건의 → 최초 **사액서원** (국가 공인·지원)
- 2019년 **유네스코 세계유산** 등재
- 기능: 선현 제사 + 후학 교육 + 사림 세력 결집

### 향약 ⭐⭐

- 사림이 향촌 운영한 자치 규약
- **4대 강목**:
  - 덕업상권 (좋은 일 서로 권함)
  - 과실상규 (잘못 서로 규제)
  - 예속상교 (예의로 교류)
  - 환난상휼 (어려울 때 서로 도움)
- **이황·이이** 모두 향약 보급에 힘씀

### 미술

- **사군자** (매·란·국·죽): 선비 절개 상징
- **신사임당 《초충도》**: 풀벌레 그림 ⭐
- **이정 (대나무)**, **황집중 (포도)** 등 사군자 화가

### 도자기

- **백자** ⭐⭐: 순백의 단순미 — 성리학적 청렴·절제 반영
  - 16세기 사림 문화의 상징
  - 분청사기(15C) → 백자(16C) 흐름
