## 고구려

- **동명성왕 (주몽)**: 건국 BC37, 졸본 / 2대 유리왕이 국내성 천도
- **태조왕**: 옥저 정복, 계루부 고씨 세습
- **고국천왕**: 왕권 강화, 행정 5부 개편, **진대법** (최초 구휼제도 → 고려 의창 → 조선 환곡)
- **미천왕**: 낙랑군 축출 **(313)**
- **고국원왕 (~371)**: 백제 근초고왕에게 평양성에서 전사
- **소수림왕**: 불교 수용 (372) + 태학 설립 (372) + 율령 반포 (373)
- **광개토대왕**:
  - 연호 **영락** 사용
  - 후연·숙신·동부여 복속
  - 신라 구원 (왜 격퇴) → **호우명그릇** (신라 내물왕 무덤 출토, 고구려-신라 관계 증거)
  - 광개토대왕릉비는 **장수왕**이 건립 (414)
- **장수왕**:
  - **평양 천도 (427)** → 남진정책 본격화
  - 도림 스파이 → **한성 함락 (475)**, 개로왕 전사 → 백제 웅진 천도
  - **중원고구려비 (충주고구려비)**: 한강 유역 장악 증거
- **문자왕 (491~519)**: 부여 완전 복속 → 고구려 최대 판도 완성
- **영양왕**:
  - **살수대첩**: 수나라 침입 격퇴, **을지문덕** — 여수장우중문시
  - 온달 아차산에서 전사
  - 이문진 — **《신집》** 편찬
- **영류왕**: 천리장성 축조, 연개소문에게 피살 (642)
- **보장왕 (~668)**:
  - 연개소문 실권 장악 (**대막리지**)
  - 도교 장려
  - **안시성 전투** — 당 태종 침입 격퇴
- **고구려 멸망 (668)**:
  - 연개소문 사후 아들들 내분 → 나당연합군
  - 평양성 함락, 보장왕 항복
  - 고구려 부흥운동: **검모잠, 안승**

## 백제

**흐름**: 고이왕 → 근초고왕 → 침류왕 → 개로왕 → 무령왕 → 성왕 → 무왕 → 의자왕

- **고이왕**: 율령, 6좌평, 16관등, 관복 (자·비·청), 한강 지배 강화
- **근초고왕**: 고국원왕 전사시킴, 마한·요서·산둥 진출, 왜에 칠지도 하사, 박사 고흥 — **《서기》** 편찬
- **침류왕**: 동진의 **마라난타**를 통해 불교 수용 (384)
- **개로왕**: 장수왕에게 한성 함락 (475), 전사 → **웅진 천도**
- **무령왕**:
  - 웅진 시대 중흥
  - **22담로**에 왕족 파견 (지방 통제)
  - 고구려 격파 (한강 일부 회복)
  - **무령왕릉**: 중국 남조 양식 벽돌무덤, 금제관식·석수 출토, 지석으로 무덤 주인 확인
- **성왕**:
  - **사비 천도 (538)**, 국호 **남부여**
  - 나제동맹으로 한강 수복 → 진흥왕에게 배신당함
  - **관산성 전투 (554)** 전사
  - 노리사치계 — 일본에 불교 전파
- **무왕**:
  - 익산 **미륵사** 창건
  - **서동설화** (무왕 = 서동)
  - **미륵사지석탑**: 현존 최고(最古)·최대 석탑
  - 해체 복원 중 금제사리봉안기 발견 → 왕비 = 사택씨 딸 (선화공주설 뒤집힘)
- **의자왕**:
  - **해동증자** 칭호, 신라 대야성 함락
  - 3천궁녀 낙화암 (과장된 설화)
- **백제 멸망 (660)**:
  - 나당연합군 (신라 김유신 + 당 소정방)
  - **황산벌 전투** — 계백 결사대 5천
  - 사비성 함락, 의자왕 항복
  - 백제 부흥운동: **복신·도침·흑치상지**
  - **백강 전투 (663)** — 왜 지원 → 나당에 패배, 부흥운동 실패

## 신라

**왕 칭호 변천**: 거서간 → 차차웅 → 이사금 → **마립간** (내물왕) → **왕** (지증왕)

- **박혁거세**: 건국 BC57, 거서간, 6부 연합
- **내물마립간**:
  - 김씨 왕위 세습 확립
  - 마립간 칭호 사용
  - 광개토대왕에게 왜구 격퇴 도움 받음 → 고구려 영향력 下
  - 부수효과로 금관가야 쇠퇴
- **눌지왕 (417)**: **나제동맹** 체결, 박제상 설화
- **지증왕 (500)**:
  - 국호 **신라** 확정
  - **왕** 칭호 사용
  - **우산국 복속** — 이사부
  - 순장 금지, 우경 장려, 동시전 설치
- **법흥왕 (514)**:
  - 율령 반포
  - 불교 공인 (527) — **이차돈** 순교
  - 골품제 정비
  - 연호 **건원** (신라 최초 독자 연호)
  - **상대등** 설치
  - **금관가야 병합 (532)**
- **진흥왕 (540)**:
  - 한강 유역 장악 (나제동맹 파기)
  - **대가야 병합 (562)** — 이사부·사다함
  - 함경도 진출 (황초령·마운령)
  - **순수비**:
    - 단양 적성비 (한강 상류) ← 진흥왕 때 세운 비석이나 순수비 아닌 전공비
    - 북한산 순수비 (한강 하류)
    - 황초령 순수비 (함경도)
    - 마운령 순수비 (함경도)
  - **화랑도** 개편 — 원광법사 **세속오계**
  - 황룡사 건립
  - 거칠부 — **《국사》** 편찬
  - 연호: 개국·대창·홍제
- **선덕여왕 (632)**:
  - 신라 최초 여왕
  - **첨성대** 건립
  - **황룡사 9층 목탑** — 자장율사 건의
  - 모란꽃 그림·개구리 소리·죽음 예언 설화
- **진덕여왕 (647)**:
  - 김춘추 당나라 외교 → **나당동맹** 본격 추진
  - **집사부** 설치 (왕명 출납 기관, 왕권 강화)
- **무열왕 (654)**:
  - **최초 진골 출신 왕**
  - 나당동맹 완성
  - **백제 멸망 (660)**
  - 김유신
