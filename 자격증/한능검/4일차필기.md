# 조선 후기 — 정치

## 흐름 ⭐⭐⭐

붕당정치 → 환국 → 탕평책 → 세도정치
(선조~현종) (숙종) (영조·정조) (순조~철종)

---

## 붕당의 형성과 분화 ⭐⭐⭐

### 붕당 형성 (선조, 1575)

- **이조전랑 자리** 다툼이 발단
  - 심의겸(기성 사림) vs 김효원(신진 사림)
- **서인**: 심의겸 지지, 서울 서쪽 거주 → 이이·성혼 계열
- **동인**: 김효원 지지, 서울 동쪽 거주 → 이황·조식 계열

### 동인의 분화 (정여립 사건 이후, 1589)

- **남인**: 온건파 — 이황 계열, 영남 기반
- **북인**: 강경파 — 조식 계열

### 광해군 때

- **북인 집권** (대북파 중심)
- 인조반정으로 북인 몰락 → 이후 역사에서 거의 사라짐

### 인조반정 이후

- **서인 집권** 장기화
- 서인 내부 분화:
  - **노론**: 송시열 중심, 강경파
  - **소론**: 윤증 중심, 온건파

## 광해군 (1608~1623) ⭐⭐⭐

| 정책                  | 내용                                               |
| --------------------- | -------------------------------------------------- |
| **중립 외교**         | 명 쇠퇴 + 후금 성장 → 실리 외교                    |
| **강홍립 투항**       | 명 요청으로 파병 → 후금에 의도적 투항              |
| **대동법**            | 경기도 선 시행 (이원익 건의)                       |
| **《동의보감》 완성** | 허준 저술                                          |
| **기유약조 (1609)**   | 임란으로 끊긴 일본 국교 재개, 동래 1포·세견선 20척 |
| **통신사 파견**       | 일본 에도막부에 파견, 조선 문화 전파               |
| **영창대군 사사**     | 선조의 적자 제거                                   |
| **인목대비 폐비**     | 선조의 왕비를 서궁에 유폐                          |

> 💡 **통신사 vs 수신사**
>
> - **통신사**: 조선 → 일본 (에도막부 시대, 조선 문화 전파)
> - **수신사**: 강화도조약 이후 (1876~) 일본 근대 문물 시찰 목적
> - **연행사**: 조선 → 청나라 (북경, 청 문물 수용)

### 인조반정 (1623)

- **서인(노론·소론)** 주도
- 명분: 중립외교 = 친명 배반, 영창대군 사사·인목대비 폐비 = 패륜
- 실제: 서인의 권력 탈환

## 인조 (1623~1649) ⭐⭐⭐

| 사건                     | 내용                                            |
| ------------------------ | ----------------------------------------------- |
| **친명배금 정책**        | 후금 적대 → 호란의 직접 원인                    |
| **이괄의 난 (1624)**     | 인조반정 논공행상 불만 → 반란, 인조 공주 피난   |
| **정묘호란 (1627)**      | 후금 침입, 인조 강화도 피신, **형제 관계** 강화 |
| **병자호란 (1636)**      | 청 태종 직접 침입, 남한산성 45일 항전           |
| **삼전도의 굴욕 (1637)** | **삼배구고두례** (3번 절·9번 머리 조아림)       |
| **소현세자·봉림대군**    | 청에 볼모로 잡혀감                              |
| **영정법 (1635)**        | 전세 고정 — 1결당 4~6두                         |

> 💡 **소현세자**: 청에서 서양 문물·천주교 접함 → 귀국 후 의문사
> → 봉림대군이 왕위 계승 = **효종**

## 효종 (1649~1659) ⭐⭐⭐

### 북벌론

- 병자호란 치욕 → 청나라에 복수하겠다는 주장
- **송시열** 중심의 노론 지지
- **윤휴** (남인)도 북벌 주장 — but 방법론 달라 대립
- 현실적으로 실행 못함 (효종 갑자기 사망)
- 의미: 실질적 군비 강화 + 왕권 강화 수단

### 나선정벌 (1654, 1658) ⭐⭐

- **나선(羅禪)** = 러시아
- 청나라가 러시아 남하를 막기 위해 조선에 조총 부대 파견 요청
- 1차 (1654): **변급** 이끄는 조총 부대 파견
- 2차 (1658): **신유** 이끄는 조총 부대 파견
- 의미: 북벌 외치면서 청을 도운 아이러니 + 조선 조총 실력 증명

---

## 현종 (1659~1674) — 예송논쟁 ⭐⭐⭐

### 예송논쟁이란?

- 효종 사망 후 **자의대비(인조의 계비)**가 상복을 몇 년 입을지를 둘러싼 논쟁
- 겉: 예법 문제 / 속: **왕권 vs 신권**, 서인 vs 남인 권력 다툼

### 1차 기해예송 (1659, 현종 즉위년)

| 구분 | 서인(송시열)                 | 남인(허목·윤휴)            |
| ---- | ---------------------------- | -------------------------- |
| 주장 | **1년** (기년복)             | **3년**                    |
| 근거 | 효종은 차남 → 차남 규정 적용 | 효종은 왕 → 왕에 맞는 예법 |
| 결과 | **서인 승리** → 1년          |
| 의미 | 왕도 예법 앞에 평등 (신권↑)  | 왕은 특별 (왕권↑)          |

### 2차 갑인예송 (1674, 현종 말년)

- 효종의 **왕비(인선왕후)** 사망 → 자의대비 상복 기간 문제
  | 구분 | 서인 | 남인 |
  |------|------|------|
  | 주장 | **9개월** | **1년** |
  | 결과 | **남인 승리** → 1년 |
  | 이후 | 남인 집권 |

> 💡 **암기법**: 기해(1차) = 서인 승, 갑인(2차) = 남인 승

## 숙종 (1674~1720) — 환국 ⭐⭐⭐

### 환국(換局)이란?

- 왕이 특정 붕당을 한꺼번에 교체하는 정치
- 공존·타협 없음 → **일당 전제화**
- 집권당이 바뀔 때마다 반대파 대거 처형·유배

### 3번의 환국

| 환국         | 연도 | 내용                                                                            | 결과                   |
| ------------ | ---- | ------------------------------------------------------------------------------- | ---------------------- |
| **경신환국** | 1680 | 남인 허적·윤휴 역모 → 숙종이 남인 축출                                          | 서인 집권              |
| **기사환국** | 1689 | **장희빈** 아들(훗날 경종) 세자 책봉 문제 → 반대한 서인 축출, **인현왕후 폐비** | 남인 집권, 장희빈 왕비 |
| **갑술환국** | 1694 | 인현왕후 복위 → 남인 축출, **장희빈 사사**                                      | 서인 재집권            |

> 💡 **장희빈**: 기사환국 때 왕비 → 갑술환국 때 사약
> 이후 서인이 **노론·소론**으로 분화

### 숙종의 주요 정책

| 정책                           | 내용                                                            |
| ------------------------------ | --------------------------------------------------------------- |
| **상평통보 전국 유통** ⭐⭐    | 조선 최초 전국 유통 화폐                                        |
| **백두산정계비 (1712)** ⭐⭐⭐ | 청과 국경 확정 — "서쪽은 압록강, 동쪽은 토문강"                 |
|                                | → 나중에 간도 귀속 문제로 분쟁 (토문강 해석 차이)               |
|                                | → 1909년 **간도협약**으로 청에 넘어감                           |
| **안용복 사건** ⭐⭐⭐         | 일본에 건너가 울릉도·독도가 조선 영토임을 확인                  |
|                                | → 일본은 1905년 러일전쟁 중 독도 불법 편입                      |
|                                | → 대한제국 **칙령 41호 (1900)**: 독도가 우리 영토임을 공식 선포 |

## 경종 (1720~1724)

- 장희빈 아들
- 재위 짧음
- **노론 vs 소론** 대립 (노론: 연잉군(영조) 세제 책봉 주장 / 소론: 반대)
- 소론이 노론 탄압 (신임사화)

## 영조 (1724~1776) ⭐⭐⭐ — 탕평책

### 탕평책

- 노론·소론·남인 고루 등용 → 붕당 갈등 완화
- **탕평비** 건립: 성균관 앞에 세워 탕평 의지 표명
- **산림(山林) 인정 안 함**: 재야 학자들이 붕당 이끄는 것 배제
- **서원 대폭 정리**: 붕당 근거지 서원 축소 (흥선대원군 때 추가 정리)
- **이조전랑 권한 약화**: 붕당 형성 원인 제거

### 주요 업적

| 정책                     | 내용                                                                               |
| ------------------------ | ---------------------------------------------------------------------------------- |
| **균역법 (1750)** ⭐⭐⭐ | 군포 2필 → **1필**로 감소, 부족분 보충: **결작**(토지세 추가)·선무군관포·어장세 등 |
| **《속대전》 편찬**      | 경국대전 이후 법령 보완                                                            |
| **신문고 부활**          | 백성 직소 제도                                                                     |
| **가혹한 형벌 폐지**     | 압슬형(무릎에 돌올리기)·낙형 폐지                                                  |
| **청계천 준설**          | 수도 치수 사업                                                                     |

### 사도세자 사건 (임오화변, 1762) ⭐⭐⭐

- 영조가 **뒤주**에 아들 사도세자를 가둬 죽임
- 배경: 노론의 사도세자 모함 + 세자의 기행
- 이후 사도세자 지지파 = **시파** / 영조 결정 지지파 = **벽파**
- 정조(사도세자 아들) 즉위 후 노론 벽파와 갈등의 씨앗

## 정조 (1776~1800) ⭐⭐⭐ — 탕평책 강화

### 왕권 강화 정책

| 정책                       | 내용                                                                                 |
| -------------------------- | ------------------------------------------------------------------------------------ |
| **규장각 설치** ⭐⭐⭐     | 왕립 도서관 + 친위 학술기관, 서얼 출신 **검서관** 등용 (이덕무·박제가·유득공·서이수) |
| **초계문신제** ⭐⭐⭐      | 37세 이하 젊은 관리 재교육 → 왕 친위 세력 육성                                       |
| **장용영 설치** ⭐⭐       | 왕 친위 군사 기구 (기존 군영 견제)                                                   |
| **수원 화성 건설** ⭐⭐⭐  | 정약용의 **거중기** 활용, 신도시 건설 — 상업 육성 + 정치 개혁 의도                   |
| **신해통공 (1791)** ⭐⭐⭐ | **금난전권 폐지** — 시전 상인 독점권 제거 → 자유 상업 허용, 사상 성장                |
| **《대전통편》 편찬**      | 법전 정비 (흥선대원군 때 《대전회통》)                                               |
| **화성 능행**              | 아버지 사도세자 능 참배 + 수원 화성 행차                                             |

> 💡 **정약용**: 정조의 핵심 브레인
> 규장각 검서관 → 수원 화성 설계(거중기 제작) → 유배 후 실학 집대성

## 세도정치 (순조~철종, 1800~1863) ⭐⭐⭐

### 특징

- 정조 사망 후 어린 왕 즉위 → 외척 가문이 권력 독점
- **안동 김씨·풍양 조씨** 등 소수 가문 장악

### 권력 구조

- **비변사** 장악 (중종 3포왜란 때 설치 → 임진왜란으로 강화 → 세도정치 최고 기관)
- **매관매직** 성행: 관직을 돈으로 삼
- 삼정(전정·군정·환정) 문란 심각

### 삼정문란 ⭐⭐⭐

| 구분             | 문란 내용                                                       |
| ---------------- | --------------------------------------------------------------- |
| **전정(토지세)** | 황무지·강바닥에도 세금 부과                                     |
| **군정(군역)**   | 죽은 자·어린아이에게도 군포 징수 (백골징포·황구첨정)            |
| **환정(환곡)**   | 봄에 빌려주고 가을에 거둬야 하는데 고리대로 변질, 억지로 떠넘김 |

### 세도정치기 민란

| 민란                           | 연도 | 내용                                                                                          |
| ------------------------------ | ---- | --------------------------------------------------------------------------------------------- |
| **홍경래의 난 (1811)** ⭐⭐⭐  | 순조 | 평안도(서북 지방) — 지역 차별 + 세도정치 반발, 홍경래 주도, 5개월 항전                        |
| **임술농민봉기 (1862)** ⭐⭐⭐ | 철종 | 진주 중심 전국 확산, 삼정문란 원인, **박규수** 안핵사 파견, **삼정이정청** 설치 (효과 없었음) |

---

## 비변사 변천 ⭐⭐⭐

설치(중종, 3포왜란 1510) → 을묘왜변(명종)으로 강화
→ 임진왜란으로 최고 기관화
→ 세도정치기 소수 가문이 장악 (의정부 유명무실)
→ 흥선대원군이 폐지 (의정부 기능 부활)

# 조선 후기 — 경제·사회

## 농업 변화 ⭐⭐⭐

### 이앙법(모내기법) 전국 확산

- 이전: 직파법 (씨앗 바로 뿌림)
- **이앙법**: 모판에서 키운 모를 논에 옮겨 심기
- 효과:
  - 노동력 절감 → **광작** 가능 (한 농민이 넓은 땅 경작)
  - 벼+보리 **이모작** 가능
  - 생산량 증가
- 부작용: 농민 분화 (일부는 광작으로 부농 / 다수는 임노동자로 전락)

### 상품 작물 재배

- **인삼·담배·채소·면화** 등 시장 판매 목적 재배
- 상업과 농업의 연결

### 지대 변화

- **타조법**: 수확량의 절반을 지대로 냄 (병작반수)
- **도조법**: 미리 정해진 양만 냄 → 풍년엔 농민 유리 → 농민 경영 의식 향상

## 수취 체제 변화 ⭐⭐⭐ (3대 개혁 비교)

| 구분     | 기존 문제          | 개혁       | 시기        | 내용                                            |
| -------- | ------------------ | ---------- | ----------- | ----------------------------------------------- |
| **전세** | 과전법→공법→불안정 | **영정법** | 인조        | 풍흉 무관 **1결당 4~6두** 고정                  |
| **공납** | 방납의 폐단        | **대동법** | 광해군~숙종 | 특산물 → **1결당 12두(쌀)** 납부, **공인** 등장 |
| **군역** | 군포 2필 과중      | **균역법** | 영조        | **1년 1필**로 감소, 부족분: 결작·선무군관포     |

### 대동법의 의의 ⭐⭐⭐

- **공인(貢人)** 등장: 관청에 물품 납품하는 상인
- 공인이 시장에서 대량 구매 → **상품 화폐 경제 발달**
- 화폐 수요 증가 → **상평통보 유통** 촉진
- 시험에 "대동법의 결과로 나타난 것" 단골 출제 → **공인 등장, 상품 화폐 경제 발달**

---

## 상업 발달 ⭐⭐⭐

### 화폐

- **상평통보 (숙종)**: 전국 유통, 조선 최초 실질적 통화

### 상인 유형

| 상인           | 특징                                                    |
| -------------- | ------------------------------------------------------- |
| **시전 상인**  | 한양 관허 상인, 금난전권 보유 → 신해통공(1791)으로 폐지 |
| **공인**       | 대동법으로 등장, 관청 납품 상인                         |
| **사상(私商)** | 허가 없는 자유 상인 — 경강상인·송상·만상·내상           |
| **송상**       | 개성 상인, 전국 유통망, **인삼** 판매                   |
| **만상**       | 의주 상인, 대청 무역                                    |
| **내상**       | 동래 상인, 대일 무역                                    |
| **경강상인**   | 한강 중심, 운송·미곡 거래                               |
| **보부상**     | 장시를 돌아다니는 행상                                  |

### 상업 시설

- **장시**: 5일장 (전국 1,000여 개로 확대)
- **포구**: 선상·객주·여각이 활동하는 상업 거점
- **도고**: 특정 물품 독점 매매하는 도매 상인

### 경영 방식

- **선대제**: 상인이 수공업자에게 원료·자금 미리 지급 → 제품 납품받음 (자본주의 맹아)
- **덕대**: 광산 전문 경영인 (광주와 계약, 노동자 고용 — 경영과 자본 분리)

---

## 신분제 동요 ⭐⭐⭐

### 양반 증가

- **공명첩**: 돈 내고 관직·신분 구매
- **납속책**: 국가 재정 부족 → 곡식 바치고 신분 상승
- **족보 위조·매매**: 양반 행세

### 노비 감소

- 군공·납속으로 해방
- 도망 증가 (국가가 찾기 어려워짐)
- **순조 때 공노비 해방 (1801)** ⭐⭐

### 중간 계층 성장

- 서얼의 관직 진출 확대 (정조 때 규장각 검서관)
- 중인·역관의 경제력 성장

### 평등 의식 성장

- **서학(천주교)** — 평등 사상 전파
- **동학** — 인내천 (사람이 곧 하늘)

---

## 천주교 (서학) ⭐⭐⭐

- 17세기 **연행사**를 통해 청에서 서적 형태로 전래
- 처음엔 학문(서학)으로 연구 → 점차 신앙으로 발전
- 18세기 말 **이승훈** 베이징에서 세례 받고 귀국 → 최초 영세
- **박해**:
  - **신해박해 (1791)**: 윤지충이 제사 거부 → 처형 (최초 박해)
  - **신유박해 (1801, 순조)**: 이승훈·정약용 형제 등 대규모 박해 ⭐⭐
  - **기해박해 (1839)**: 프랑스 선교사 처형
  - **병인박해 (1866, 흥선대원군)**: 최대 규모 → 프랑스 **병인양요** 원인 ⭐⭐⭐

---

## 동학 ⭐⭐⭐

- **최제우** (1860년 창시)
- **인내천(人乃天)**: 사람이 곧 하늘 → 신분 평등 사상
- **시천주**: 모든 사람 안에 하늘님이 있다
- 동학 = 서학(천주교)에 대항하는 민족 종교
- 유·불·도 + 민간 신앙 혼합
- 최제우 처형 (1864) → 2대 교주 **최시형**
- 이후 **동학농민운동 (1894)** 으로 이어짐

# 조선 후기 — 문화·사상

## 성리학의 변화

### 양명학 ⭐⭐

- 중국 왕양명이 주창
- **지행합일(知行合一)**: 알고 실천하는 것이 하나

- **정제두** (강화도 거주) → **강화학파** 형성

## 도자기 변천 ⭐⭐

고려 청자 → 조선 전기 분청사기 → 조선 전기 백자
→ 조선 후기 청화백자(푸른 그림 그린 백자) → 달항아리

## 조선 후기 흐름 총정리

광해군(중립외교·대동법·인조반정)
→ 인조(친명배금·정묘·병자호란·삼전도)
→ 효종(북벌론·나선정벌)
→ 현종(예송논쟁 1·2차)
→ 숙종(환국 3번·상평통보·백두산정계비·안용복)
→ 영조(탕평·균역법·속대전·사도세자)
→ 정조(규장각·초계문신·장용영·화성·신해통공)
→ 세도정치(순조~철종·삼정문란·홍경래·임술봉기)
→ 흥선대원군(개혁)→고종→개항
